# ナゾヨミ

写真に写った独自記号を、端末内の画像処理だけでひらがなへ変換するWebアプリです。画像や解析結果をサーバーへ送信しません。

## 使い方

1. 「撮影または画像を選択」から問題画像を開く
2. 4つの角を問題文の外側へ合わせる
3. 「解析する」を押す
4. 誤認識があれば画像上の枠を選択して修正する
5. ひらがなをコピーする

対応範囲は、横書きの清音、濁点、半濁点、「ん」、「？」です。強い反射、ぼけ、記号の欠損がある画像は手動修正が必要です。

## 開発

```bash
pnpm install
pnpm dev
pnpm test
pnpm build
```

- `src/domain.ts`: 50音対応表、濁点変換、文字列復元
- `src/vision/`: 台形補正、輪郭抽出、テンプレート照合
- `src/main.ts`: 撮影、範囲補正、認識結果編集

OpenCV.jsは初回解析時に遅延ロードされます。Viteの成果物はGitHub Actionsからリポジトリの `/nazonazo/` へ配備されます。
