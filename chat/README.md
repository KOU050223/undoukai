# chat

同一WiFi（LAN）内の別端末とリアルタイムに通信できる、画像アップロード対応チャットツール。

- フロント: Vite + React + TypeScript
- 通信: WebSocket サーバー（Node.js / `ws`）でブロードキャスト

## 起動

```bash
npm install
npm run dev
```

`npm run dev` でフロント（Vite, ポート5173）と WebSocket サーバー（ポート3001）が同時に立ち上がります。

## 別端末から接続する

1. サーバーを起動したPCの LAN IP（例: `192.168.x.x`）を確認する
   ```bash
   ipconfig getifaddr en0   # macOS
   ```
2. 同じWiFiにつながった別端末のブラウザで以下を開く
   ```
   http://192.168.x.x:5173
   ```
   WebSocket の接続先はアクセス中のホスト名から自動決定されるため、追加設定は不要です。

## 機能

- テキスト送信（Enter または送信ボタン）
- 画像アップロード（📷ボタン、data URLとして送信・表示）
- 名前の変更（localStorageに保存）
- 接続状態表示・自動再接続
- 直近100件の履歴を新規接続時に受信

## 注意

- 画像はサーバーのメモリ上に保持（永続化なし）。サーバー再起動で履歴は消えます。
- LAN内利用を想定。インターネット公開向けの認証・暗号化はありません。
